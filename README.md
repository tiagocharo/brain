# Brain
Brain.js is a library of Neural Networks written in JavaScript.

**First step**
### `git clone https://github.com/tiagocharo/brain.git`

**Second step**
### `cd brain`

**Third step**
### `npm install`

**Fourth step**
### `node index.js`

## Learn More

You can learn more in the [brain.js](https://github.com/BrainJS/brain.js).
